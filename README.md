# Linko

Linko is a mini social network web application built with Angular and Django REST Framework.

## Features (planned)
- User authentication (JWT)
- Create, edit, delete posts
- Comments and likes
- User profiles

## Tech Stack
- Frontend: Angular
- Backend: Django + DRF

## Project Structure
- /backend - Django REST API
- /frontend - Angular application
